## Zhihu
### ZhihuCollection_Image
爬取知乎收藏夹中所有回答的图片；
涉及到断点续传和html网页解析
爬取网页链接示例：https://www.zhihu.com/collection/29997839
### dazhong
大众点评中某家餐厅用户评价的图片
涉及到代理使用和网页解析
注：代码中的代理并不好用，测试时将涉及到代理的部分去掉即可，但需要放慢抓取速度，防止封ip
### proxies

但免费的代理并不好用，验证某些网站可用，用到正在要爬取的网站，可能也是访问不了的
